    1. Подключить фреймверк по типу Bootstrap, Foundation, etc и создать 2 колонки (sidebar + content)
      В сайдбаре будут фильтры:
      - чекбоксы для всех возможных цветов продуктов
      - селект с ценой (Price ASC, Price DESC)
      В контенте будут карточки товаров - картинка товара, название, цвет и цена
    2. Шлем AJAX на http://localhost:3000/products и забираем все продукты
      Дальше рендерим их в контенте, а так же строим фильтры по цветам в сайдбере
    3. По клику на чекбокс с определенным цветом показываем только продукты этого цвета
    4. По выбору селекта с ценой сортируем продукты по возрастанию или убыванию цены, при чем фильтры с цветом должны сохраняться
